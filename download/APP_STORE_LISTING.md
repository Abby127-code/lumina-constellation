# Lumina Constellation — App Store 上架指南

## 上架时间表

| 步骤 | 预计时间 | 依赖 |
|------|---------|------|
| 1. 域名 + SSL | 1 天 | 购买域名 |
| 2. 部署 Web 版到 Vercel | 1 小时 | 域名就绪 |
| 3. 注册 LianLian 商户 | 3-5 天 | 营业执照 |
| 4. Google Play 开发者账号 | 1 天（审核快） | $25 一次性 |
| 5. Apple 开发者账号 | 2-7 天（审核慢） | $99/年 |
| 6. Capacitor 打包 Android APK | 1 天 | Web 部署就绪 |
| 7. Capacitor 打包 iOS IPA | 1 天 | Apple 账号 + macOS |
| 8. Google Play 上架审核 | 1-3 天 | APK 就绪 |
| 9. Apple App Store 审核 | 1-7 天 | IPA 就绪 |
| **总计** | **7-14 天** | |
